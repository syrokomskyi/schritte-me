It describes the process of extracting the text from photo with preserving original structure and formatting.

You can change the instructions (see below) as you wish.

## Process

Open the ChatGPT.
Attach a photo to your message.
Paste the prompt below.
Choose the model `o4` or `o3`.

## Prompt

```
v1.0.0

You will be given an image containing formatted text (headings, paragraphs, bullet points, tables, etc.).

Manually extract the text from the photo.

Extract **all** the text exactly as it appears. Leave the original text as well.

Preserve the original structure and formatting - including line breaks, indentation, lists, headings, bold/italic styling, and tables.

Do not solve the tasks.

Output the result in Markdown (or plain text) so that the layout reflects the source image as closely as possible.

```
