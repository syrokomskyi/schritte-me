# Arbeitsbuch 116 UK
| Original                                                                               | UK                                                                              |
| -------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------- |
| **Markieren Sie und ergänzen Sie Pfeile.**                                             | **Позначте та доповніть стрілки.**                                              |
| **Grammatik entdecken**                                                                | **Відкрийте граматику.**                                                        |
| Ihr Vater lebt in Westungarn.                                                          | Її батько живе у Західній Угорщині.                                             |
| Ihr Hobby ist Tanzen.                                                                  | Її хобі — танці.                                                                |
| Ihre Eltern sind geschieden.                                                           | Її батьки розлучені.                                                            |
| Sein Hobby ist Badminton spielen.                                                      | Його хобі — гра в бадмінтон.                                                    |
| Seine Frau heißt Lan.                                                                  | Його дружину звуть Лан.                                                         |
| Seine Kinder leben in Berlin.                                                          | Його діти живуть у Берліні.                                                     |
| **Ergänzen Sie: sein – seine – ihr – ihre.**                                           | **Доповніть: sein – seine – ihr – ihre.**                                       |
| Das ist meine Freundin Basima aus Syrien. Ich kenne sie aus dem Schwimmbad.            | Це моя подруга Басіма з Сирії. Я знаю її з басейну.                             |
| Schwimmen ist ihr Hobby.                                                               | Її хобі — плавання.                                                             |
| Sie ist verheiratet und sie hat zwei Kinder:                                           | Вона заміжня і має двох дітей:                                                  |
| Tochter ist zehn Jahre alt und Sohn ist acht.                                          | Дочка десяти років, а син — восьми.                                             |
| Kinder schwimmen auch gern.                                                            | Діти також люблять плавати.                                                     |
| Mann Kalil schwimmt nicht so gern.                                                     | Її чоловік Каліл не дуже любить плавати.                                        |
| Hobby ist Schachspielen.                                                               | Його хобі — гра в шахи.                                                         |
| Früher haben Kalil und Vater oft zusammen Schach gespielt.                             | Раніше Каліл і батько часто грали в шахи разом.                                 |
| Aber Eltern leben nicht in Deutschland.                                                | Але батьки не живуть у Німеччині.                                               |
| Dort lebt auch Schwester.                                                              | Там також живе сестра.                                                          |
| Mann ist Amerikaner.                                                                   | Її чоловік — американець.                                                       |
| **Ergänzen Sie: -e oder -r.**                                                          | **Доповніть: -e або -r.**                                                       |
| a. Sandro kommt heute etwas später. Sein Sohn muss zum Arzt.                           | а. Сандро сьогодні прийде трохи пізніше. Його сина потрібно відвести до лікаря. |
| b. Hakan arbeitet als Koch in einem Restaurant. Sein Chef ist sehr professionell, aber | б. Хакан працює кухарем у ресторані. Його шеф дуже професійний, але             |
| sein Kollegen sind nicht besonders nett.                                               | його колеги не дуже приємні.                                                    |
| c. Vor zwei Tagen haben wir Petre besucht. Sein Wohnung ist sehr schön.                | в. Два дні тому ми відвідали Петре. Його квартира дуже красива.                 |
| d. Ilena kann nicht zum Deutschkurs kommen. Ihr Kinder sind krank.                     | г. Ілена не може прийти на курс німецької. Її діти хворі.                       |
| e. Meine Schwester hat zwei Kinder, ihr Sohn heißt Leo und ihr Tochter heißt Lena.     | д. У моєї сестри двоє дітей, її сина звати Лео, а дочку — Лена.                 |
| **Was erzählt Marina? Schreiben Sie.**                                                 | **Що розповідає Марина? Напишіть.**                                             |
| sehr nett und lustig                                                                   | дуже милий і веселий                                                            |
| ganze Familie: seit 25 Jahren in Deutschland                                           | уся сім'я: 25 років у Німеччині                                                 |
| Schwester und ihr Bruder in Deutschland geboren                                        | Сестра і її брат народилися в Німеччині                                         |
| Schwester: hat ein Restaurant                                                          | Сестра: має ресторан                                                            |
| Restaurant: am Schillerplatz                                                           | Ресторан: на Шіллерплац                                                         |
| Pizzen: sehr lecker                                                                    | Піца: дуже смачна                                                               |
| dort: Ivano treffen                                                                    | там: зустріти Івано                                                             |
| dann: heiraten                                                                         | потім: одружитися                                                               |
| **Hallo Marina, wie geht es dir?**                                                     | **Привіт, Марина, як справи?**                                                  |
| Super! Ich habe am Wochenende geheiratet.                                              | Супер! Я одружилася на вихідних.                                                |
| Wirklich? Wen denn? Erzähl mal.                                                        | Справді? На кому? Розкажи.                                                      |
| Also, sein Name ist ...                                                                | Ну, його ім'я — ...                                                             |
